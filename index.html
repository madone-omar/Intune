Script Overview

This script checks if your system meets all the necessary requirements for BitLocker silent encryption. It ensures the following conditions are satisfied:

  • TPM (Trusted Platform Module) version 1.2 or higher is available.
  • System is running in UEFI firmware mode (not Legacy BIOS).
  • Secure Boot is enabled.
  • PCR7 Configuration is ready for encryption.
  • Kernel DMA Protection is active.
  • Windows Recovery Environment (WinRE) status is verified.

TSS Log Collector

Collect TSS logs with Intune Remediation by executing the following steps:

  • Downloads the TSS tool from aka.ms/getTSS
  • Creates a TempZ folder in C:\ and copies the TSS folder, then unzips it.
  • Runs PowerShell as administrator and navigates to the Temp folder with: cd C:\Tempz\TSS
  • Executes the script using: .\TSS.ps1 -CollectLog DND_SetupReport -AcceptEula
  • Results are stored in: C:\MS_DATA
